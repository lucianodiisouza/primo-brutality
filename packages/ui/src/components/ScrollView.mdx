# ScrollView

Scrollable region with content padding and keyboard-friendly defaults.

## Usage

```tsx
import { ScrollView } from "@primo-brutality/ui";

<ScrollView contentPadding="md" className="flex-1">
  {items.map((item) => (...))}
</ScrollView>
```
