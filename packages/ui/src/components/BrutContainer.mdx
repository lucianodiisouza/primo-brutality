# BrutContainer

Full-screen cream page wrapper with padding and optional max-width column.

## Usage

```tsx
import { BrutContainer } from "@primo-brutality/ui";

<BrutContainer maxWidth="md" padding="lg">
  {children}
</BrutContainer>
```
