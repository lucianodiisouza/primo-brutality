# BrutToggle

On/off switch for settings screens, feature flags, and preference rows.

## Usage

```tsx
import { useState } from "react";
import { BrutToggle } from "@primo-brutality/ui";

const [enabled, setEnabled] = useState(false);

<BrutToggle
  value={enabled}
  onValueChange={setEnabled}
  accessibilityLabel="Push notifications"
/>
```
