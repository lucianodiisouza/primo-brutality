# BrutCard

Bordered content surface with optional hard shadow, padding, and white/cream
variants.

## Usage

```tsx
import { BrutCard, BrutHeading } from "@primo-brutality/ui";

<BrutCard shadow="hard" padding="lg">
  <BrutHeading level={3}>Dashboard</BrutHeading>
</BrutCard>
```
