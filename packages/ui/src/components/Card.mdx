# Card

Bordered content surface with optional hard shadow, padding, and white/cream
variants.

## Usage

```tsx
import { Card, Heading } from "@primo-brutality/ui";

<Card shadow="hard" padding="lg">
  <Heading level={3}>Dashboard</Heading>
</Card>;
```
