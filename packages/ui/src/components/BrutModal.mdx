# BrutModal

Centered overlay dialog. Backdrop press dismisses; inner panel stops
propagation.

## Usage

```tsx
import { BrutModal } from "@primo-brutality/ui";

<BrutModal visible={open} onClose={() => setOpen(false)} title="Confirm">
  <BrutText>Are you sure?</BrutText>
</BrutModal>
```
