# BrutScrollView

Scrollable region with content padding and keyboard-friendly defaults.

## Usage

```tsx
import { BrutScrollView } from "@primo-brutality/ui";

<BrutScrollView contentPadding="md" className="flex-1">
  {items.map((item) => (...))}
</BrutScrollView>
```
