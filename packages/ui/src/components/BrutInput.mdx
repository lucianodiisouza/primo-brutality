# BrutInput

Text field with brutal border styling, optional label, hint, and error message.

## Usage

```tsx
import { BrutInput } from "@primo-brutality/ui";

<BrutInput
  label="Email"
  placeholder="you@example.com"
  hint="We never share your email."
/>
```
