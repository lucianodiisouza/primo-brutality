# Modal

Centered overlay dialog. Backdrop press dismisses; inner panel stops
propagation.

## Usage

```tsx
import { Modal } from "@primo-brutality/ui";

<Modal visible={open} onClose={() => setOpen(false)} title="Confirm">
  <Text>Are you sure?</Text>
</Modal>
```
