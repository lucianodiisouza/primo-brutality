# Text & Heading

Typography primitives with brutal font family, size/weight presets, and semantic
colors. Use `Heading` for page titles and section headers.

## Usage

```tsx
import { Text, Heading } from "@primo-brutality/ui";

<Heading level={2}>Section title</Heading>
<Text color="muted">Supporting copy</Text>
```
